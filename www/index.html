Simple HTTP server(SHS) is a server which hosts image files, html pages and text files. Client(For example a Web Browser) can request for any information by connecting to the host webserver which is running on localhost:8097 SHS can serve multiple clients at the same time. It uses key concepts of socket multiplexing. SHS also implements persistant connection to the client and sets the timeout for the client to be 10s. If the client is idle for more than 10s without any request to the server. The server disconnects the client. The SHS project was started at the University of Colorado Boulder in September 2014.
